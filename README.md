# Senthuron-cafe
A responsive, interactive café website built with HTML5, CSS3, and Vanilla JavaScript
Senthuron Café Website
A fully responsive café website built with HTML, CSS, and JavaScript.

Features
Hero Section with video background and call-to-action buttons

Interactive Menu with category filtering (Espresso, Brew, Pastry)

Daily Specials section showcasing featured items

Contact/Details Section with address, working hours, and reservation form

Mobile Responsive design with hamburger navigation

Interactive Elements including form validation and smooth scrolling

Live Demo
[View Live Website](https://Mounisha-D.github.io/Senthuron-cafe)

Project Structure
SENTHURON-CAFE/
├── index.html          # Main HTML file
├── main.css            # All styling with CSS custom properties
├── app.js              # JavaScript functionality
├── assets/             # Image assets
Technologies Used
HTML5 - Semantic markup

CSS3 - Flexbox, Grid, Media Queries, Custom Properties

Vanilla JavaScript - No frameworks or libraries

Font Awesome - Icons

Google Fonts - Montserrat & Dancing Script

How to Run Locally
Clone this repository: git clone https://github.com/YOUR-USERNAME/Senthuron-cafe.git

Navigate to folder: cd Senthuron-cafe

Open index.html in your browser

That's it! No build process required
